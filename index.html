    About library management

    It is a console application written in C++ it is made using concepts of OOPs and file handling, in this you can do-

    1. Create/update profile of admin and student
    2. Books records
    3. Issue book/Return book
    4. Pay fine

    This is application is developed by Md Danish an student of engineering technology of IP University.

    How to use it

    1. Download the zip file and extract it any where you want.
    2. Download Dev C++, if you have already then go to next step.
    3. Then open Dev c++ go to file then open and browse the main.cpp from the folder which you have extracted from zip file.
    4. Then goto Execute in the tool bar of Dev C++ and click on compile & run;
    5. Done enjoy!
